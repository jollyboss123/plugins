# Do Work

- Direct invocation is allowed.
- `prd-to-plan` is optional support, not a mandatory prerequisite.

## Steps
1. Understand scope and target behavior.
2. Reuse existing plan if available, otherwise do lightweight planning.
3. Implement incrementally.
4. Run typecheck/test equivalents for the current project.
5. Report changes and validation results.
