# @jollyboss123/prd-workflow

PRD workflow plugin package for Codex, Claude, and Cursor.

## Install

```bash
npm install @jollyboss123/prd-workflow
```

Postinstall uses safe defaults (local install only).

## Commands

```bash
npx @jollyboss123/prd-workflow install --local
npx @jollyboss123/prd-workflow install --global
npx @jollyboss123/prd-workflow verify
```

Global install is opt-in and writes to conservative known paths only.
