# Prompt: write-a-prd

Canonical bundle: `prd-workflow`
Canonical version: `1.0.0`
Workflow ID: `write-a-prd`

## Prompt
Create a structured PRD for the feature request I provide.

Requirements:
- Explore repository context before drafting assumptions.
- Ask focused clarifying questions only for high-impact ambiguity.
- Output sections exactly:
  1. Problem Statement
  2. Solution
  3. User Stories
  4. Implementation Decisions
  5. Out of Scope
  6. Further Notes
- Keep decisions durable and avoid brittle file-level commitments unless I explicitly ask.
