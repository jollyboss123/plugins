# @jollyboss123/ralph-workloop

Bounded automation workloop plugin package for Codex, Claude, and Cursor ecosystems.

## Install

```bash
npm install @jollyboss123/ralph-workloop
```

Postinstall uses safe defaults (local install only).

## Commands

```bash
npx @jollyboss123/ralph-workloop install --local
npx @jollyboss123/ralph-workloop install --global
npx @jollyboss123/ralph-workloop verify
```

Global install is opt-in and writes to conservative known paths only.
